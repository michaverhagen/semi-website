Callout element

Callouts are used to emphasize a specific part of the page extra attention.

Example

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce commodo orci eget est condimentum, ut facilisis enim blandit.

This part of the text get's extra attention. And can also link to somewhere.

Multiple elements can exist inside this element

Mauris imperdiet ut turpis ut tincidunt.

HTML code snippet

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce commodo orci eget est condimentum, ut facilisis enim blandit.</p> <div class="callout"> <p>This part of the text get's extra attention. And can also <a href="#" title="">link to somewhere</a>.</p> <p>Multiple elements can exist inside this element</p> </div> <p>Mauris imperdiet ut turpis ut tincidunt.</p>
